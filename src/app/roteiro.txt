Programação com Bootstrap + Angular + NodeJS
* Instalar NodeJS (web)

* instalar Angular no projeto (terminal)
    npm install -g @angular/cli

* Instalar Angular Material no projeto (terminal)
    ng add @angular/material

* Criar um projeto/programa (terminal)
    ng new "nomeProjeto" --no-standalone

* Executar o projeto/programa
    ng serve -o
############  ############
** Configurações iniciais no projeto

- Excluir o conteúdo de: /src/app/app.html
    adicionar o comando: <router-outlet></router-outlet>

- Instalar o Bootstrap no projeto
    npm install bootstrap
- Adicionar biblioteca do Bootstrap no projeto: /src/style.scss
    @import '../node_modules/bootstrap/dist/css/bootstrap.min.css';

** Criar componentes no projeto
    ng generate component componentes/"nome do componente"
    ng generate component componentes/button

** Adicionar o componente button no projeto: /button/button.html

** Adicionar estilo ao componente button: /button/button.scss

** Chamar o componente button: /app/app.html
    <app-button></app-button>

#######
Configurar magem para o component button: /app/app.html

#######
Configurar o button para ser DINÂMICO: /button/button.ts
    - Criar uma variável tipo string -> label do button
    - Chamamos a variável em button.html {{txtButton}}
    - Adicionar propriedade input ao elemento button: button.ts
    - Atribuir vários valores para o label do txtButton: app.html
